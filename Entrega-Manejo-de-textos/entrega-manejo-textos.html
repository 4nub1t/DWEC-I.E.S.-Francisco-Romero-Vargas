EJERCICIOS DE MANEJO DE TEXTOS EN JAVASCRIPT BÁSICO

1. Invertir una cadena. Usar una función a la que se le pase un texto como parámetro. Usar un bucle for.

2. Contar cuántas veces aparece una letra en un texto como parámetro. Usar un bucle for.

3. Detectar si un texto es un palíndromo. Un palíndromo es una palabra o frase que suena igual leyéndola hacia adelante que hacia atrás (de manera invertida, comenzando desde la última letra de la frase). Usar una función a la que se le pase un texto como parámetro.

4. Contar las vocales en un texto. Usar una función a la que se le pase un texto como parámetro. Usar un bucle for.

5. Reemplazar una palabra en una frase. Usar una función a la que se le pase un texto como parámetro. Usar un bucle for.

6. Poner en mayúscula la primera letra de cada palabra. Usar una función a la que se le pase un texto como parámetro. Usar un bucle for.

7. Contar cuántas veces una letra aparece en una palabra usando charAt(). Usar una función a la que se le pase un texto como parámetro. Usar un bucle for.
